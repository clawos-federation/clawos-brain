# 第三章：精神到神经 —— 情绪是如何变成电信号的？

## 3.1 翻译者：大脑的“换能”逻辑

在上一章中，我们选择了“注意力锚点”。但为什么一个看不见、摸不着的“想法”，能让你的手心出汗，或者让你的心跳加速？

这就是生命最神奇的翻译过程：**换能（Transduction）**。你的精神状态（心理层）通过神经元突触，被翻译成了物理层面的电信号和化学信号。

## 3.2 情绪的“电解质”：递质流转

当你的注意力失去锚点，陷入焦虑时，大脑的“警报中心”杏仁核会瞬间激活。
- **电信号爆发**：成千上万个神经元同时放电，形成一种高频、混乱的脑波。
- **化学瀑布**：为了传导这些压力，大脑会大量释放**谷氨酸（Glutamate）**。如果缺乏有效抑制（如 GABA 递质不足），这种“过度放电”会直接灼伤神经连接。

## 3.3 传导路径：从前额叶到全身

健康的注意力锚点（外投或内观）能强化**前额叶皮层**的抑制功能。
- **降压指令**：前额叶会向迷走神经（Vagus Nerve）发送低频电脉冲。
- **全身反馈**：这个电信号像一道指令，瞬间通过脊髓传遍全身，告诉每一个脏器：*“环境安全，停止应激，开始修复。”*

## 3.4 逻辑隔离的价值

理解这一环，你就会明白：**“心态好”不是安慰，而是在优化你的神经电网。** 你的每一次专注，都是在为你的身体进行一次精密的“电路除噪”。
